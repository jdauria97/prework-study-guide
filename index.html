HTML


  • Hyper Text Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element contains meta info to determine language, usability, etc.
  • HTML is the markup language which provides structure for the application.

CSS


  • Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Can change the asthetic of individual areas or the web page as a whole.
  • <div> will create sections that can be assigned classes or id's.
  • Id's are for individual selectors.
  • Classes can be assigned to multiple selectors

Git


  • Git is a version control filing system that connects multiple machines and users to a centeral server for writing code
  • Utilizes command line utility programs to execute Unix style commands
  • git status
    checks what branch we are currently on
  • checkout -b 'branch-name'
    creates a new branch and switches to it
  • git add -A
    stages the local branch
  • git commit
    commits the changes to the local branch
  • -m
    adds a comment to the commit
  • git pull
    updates local branch to remote branch
  • git push
    pushes local branch to remote

JavaScript


  • JavaScript is a programing language that gives usability to applications.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions and variables must be defined before they are executed or called.
  • JavaScript recognizes capitalizations within code.
  • 'Strings' are primitives, 'Arrays' are groups of individual strings, numbers, objects, and/or Booleans.
  • Indeces are numbered starting with '0' and increasing by values of '1'.
